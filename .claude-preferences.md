# Claude Preferences

## Deployment Protocol
For ALL tasks:
1. **Always commit changes to GitHub** after making code changes
2. **Always deploy to Vercel** in production mode
3. **Display the live deployment URL** to the user

Commands to use:
```bash
git add <files>
git commit -m "<message>"
git push origin main
vercel --prod
```

## Task Types
This applies to:
- Bug fixes
- Feature additions
- Layout adjustments
- Styling changes
- Code refactoring
- Any code modification
